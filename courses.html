About Our Courses

Koding Kids



Language: Scratch

Recommended Age Range: 3rd-6th grade

Scratch is a block-based programming language. Never even seen a code? That’s fine! You don’t even need to write in Scratch, you just drag and drop blocks of code. We’ll help you start your Scratch journey by introducing you to all the blocks and how you can use them. At the end of the course everyone will have the opportunity to create and share their very own project. We’ll cover the basic blocks in Scratch such as Look Blocks, Sound Blocks, Event Blocks, Control Blocks, and Sensing Blocks.

Instructors:

University High School- Emily Zhang, Elaine Chao, Kevin Thai, Susanna Mathew, Vaibhav Bommisetty

Koding Kids Pro

Language: Python

Recommended Age Range: 7th-8th grade

This course is an introduction to Python, a general-use coding language. No previous experience is needed to join this class. We’ll be teaching the basics by starting at the very beginning. You’ll start by creating variables and work your way up to finally create and share your very own project using all the skills you acquire over the term of the course. We will cover conditionals, randomization, if-statements, loops, user input, and matplotlib.

Instructors:

University High School- Minh Trinh, Ethan Lai, Jack Guo, Sophia Liang, Wilson Su
